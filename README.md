# MestradoOpen

Instalar as dependências

```bash
  pip install -r script/requirements.txt
```

Rode os testes

```bash
  python -m pytest test
```

# Verificações de qualidade

para verificar a qualidade do código executar o comando
```bash
  pylint script
```
